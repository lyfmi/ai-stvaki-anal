Сигнальный Бот для Беттинг Арбитража Трафика: AI Анализ Ставок

 Скрипты для Telegram  3 812

AI Анализ Ставок - лучший сигнальный бот для беттинга арбитража трафика. Данный бот даёт возможность получить от пользователя скрин события и делает AI анализ с рекомендацией куда ставить ставку, плюс выдаёт прогнозируемую вероятность захода и краткое объяснение почему так нужно поставить ставку.

Перед этим бот просит подписаться и внести первый депозит по вашей партнёрской ссылке, таким образом гарантированно закрепляет игрока за вами. Так же бот имеет ряд других функций для создания высоких конверсий.

Как настроить и запустить бот?

Запускать бот нужно на Windows на 10 версии или желательно Windows сервере 2022 (для работы бот 24/7), инструкция как купить Windows сервер и подключиться к нему
  1. Запустите скрипт start.cmd
  2. Скрипт вам выдаст ID который нужен для активации скрипта, активировать скрипт можно в боте @GraySchemebot > Магазин > Мои покупки > Выберите скрипт > Активация > Введите ID из скрипта > Получите ключ активации > Софт активирован и только в случае если покупали у админа пишите ему для получения ключа активации, скинув ID
  3. Дополнительно напишите админу https://t.me/bivenmo и предоставьте ссылку своего Телеграм бота, для активации Web-App в вашем боте
  4. Откройте файл config.yaml и заполните следующие поля:
  • Токен бота, который нужно создать в @botfather
  • Ссылка на канал, куда будут подавать заявку
  • ID этого канал, просто перешлите любой пост из канала в боте https://t.me/GetIDUserInfo_bot для получения id (канал А)
  • ID канала (нужно создать ещё 1), куда будут приходить отчёт от партнёрской программы (канал Б)
  • ID админа бота, просто нажмите /start в боте https://t.me/GetIDUserInfo_bot для получения (бот А)
  • Ссылка на ваш Телеграм аккаунт
  • Ссылка где расположен фаил user.db (ничего не менять)
  • Включить или выключить возможность покупки безлимита в боте (ваша доп. монетизация)
  • Цена покупки безлимита в $
  • Ссылка на покупку (рекомендуется на диалог с админом с автозаполненным текстом)
  1. Далее в файле lang.yaml можете отредактировать тексты под себя, главное укажите свой промокод, в разных языках, в указанном на скрине местах:
  1. Создайте ещё одного бота в @botfathet, он нужен для отправок отчётов от партнёрки в канал, где будут храниться отчёты по пользователям для проверки регистрации и депозита (бот Б)
  2. Добавьте Бот А, в каналы А и Б, а так же, бот Б в канал Б:
  1. Пропишите постбеки для партнёрской ссылки и промокода в партнёрской программе, для этого в рекламных материалах нажмите на значёк постбека:
  1. Далее пропишите постбеки для регистрации и для первого депозита:
  • Для регистрации: https://api.telegram.org/bot6758697366:AAHqzTAXphqRg9XlHiFr2hzQ4cH/sendMessage?chat_id=-1002338203752&text={sub1}
  • Для первого депозита: https://api.telegram.org/bot6758697366:AAHqzTAXphqRg9XlHiFr2hzQ4cH/sendMessage?chat_id=-1002338203752&text={sub1}|Firstdep|{amount}
Вот пример как нужно указать:
  1. Далее снова запустите start.cmd для запуска работы бота и войдите в него с аккаунта админа, нажмите старт, выберите язык и введите /admin4, далее укажите свою партyёрскую ссылку в формате: 
  1. После чего бот настроен и готов к работе, рекомендую протестировать все функции перед тем как заливать трафик.

Как админу перейти в раздел игр без регистрации и пополнения?

  1. Для этого нужно взять первый постбек и вместо {sub1} указать свой телеграм id и перейти по ссылке постбека в браузере.
  2. Затем взять второй постбек и указать свой id как и в первом случае, а вместе {amount} указать любую сумму к примеру 777 и так же перейти в по ссылке
